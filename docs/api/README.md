# API Docs

## Purpose

Documents public and internal API contracts.

Implemented contracts cover auth, user, KYC, ledger, wallet, limits, and fee services.

## Owner

API platform and service owners.

## What Should Go Inside

- OpenAPI specs
- API versioning rules
- Error code catalog references
- Idempotency requirements
- Provider callback contract docs

## What Should Not Go Inside

- Secrets or access tokens
- Private provider credentials
- Unversioned breaking changes
- Database schema internals unless relevant to API contract
