# Compliance Docs

## Purpose

Stores compliance artifacts for audit, privacy, retention, KYC, admin access, and financial reporting.

## Owner

Compliance and security engineering.

## What Should Go Inside

- PII data maps
- Audit event catalog
- Data retention policy
- Admin access review checklist
- KYC document protection policy

## What Should Not Go Inside

- Raw KYC documents
- Full identity numbers
- Secrets
- Unapproved financial reports with customer data
