# Load Tests

## Purpose

Contains performance and capacity tests for critical APIs and workflows.

## Owner

SRE and quality engineering.

## What Should Go Inside

- Login spike tests
- Ledger posting throughput tests
- Wallet transfer throughput tests
- Provider callback burst tests
- Reconciliation ingestion tests

## What Should Not Go Inside

- Production-impacting load plans without approval
- Secrets
- Real customer data
- Functional unit tests
