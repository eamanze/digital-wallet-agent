# ECS Module

## Purpose

Creates ECS Fargate clusters, services, task definitions, and load balancer integration.

## Owner

Cloud platform engineering.

## What Should Go Inside

- ECS cluster and service resources
- Task definitions
- Autoscaling
- Health check wiring
- Task IAM roles

## What Should Not Go Inside

- Application source code
- Plaintext secrets
- Database migrations
- Provider business logic
