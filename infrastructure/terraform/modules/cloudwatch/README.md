# CloudWatch Module

## Purpose

Creates logging, metrics, alarms, and dashboards for AWS resources and services.

## Owner

SRE and cloud platform engineering.

## What Should Go Inside

- Log groups
- Metric alarms
- Dashboards
- Retention policies
- Alert integration wiring

## What Should Not Go Inside

- Sensitive log payloads
- Application source code
- Audit source-of-truth records
- Secrets
