# Ops Scripts

## Purpose

Contains controlled operational helper scripts for support, diagnostics, and recovery.

## Owner

SRE and operations platform team.

## What Should Go Inside

- Read-only diagnostics
- Runbook helpers
- Safe reconciliation validation helpers
- Deployment verification scripts

## What Should Not Go Inside

- Unapproved production mutation scripts
- Direct ledger editing
- Secrets
- Scripts that bypass maker-checker controls
